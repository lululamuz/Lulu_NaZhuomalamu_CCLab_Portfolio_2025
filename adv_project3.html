Experimental Camera

Advanced CC Lab · Project 3 · p5.js · Teachable Machine

Live Sketch

Description

This project explores the camera as an interactive communication tool rather than a device for simple recording. Using Teachable Machine, the system recognizes several trained hand gestures and translates them into real-time subtitles on screen.

I wanted to imagine a more expressive and accessible camera interface: one that responds to gesture, language, and presence. The project also reflects on communication beyond spoken words, turning sign-based input into a cinematic live subtitle experience.

Interaction

Show one of the trained gestures to the webcam: hi, yes, or love.

Keyboard backup: 1 = HELLO, 2 = YES, 3 = LOVE
